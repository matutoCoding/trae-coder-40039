## 1. 产品概述

轧钢厂热轧带钢业务管理后台，用于轧钢车间对加热、轧制和卷取全流程进行数字化管理。系统覆盖板坯入炉到成品打捆称重的完整生产链路，帮助车间实现生产数据实时监控、工艺参数追溯、质量检测记录等核心能力。

- 目标用户：轧钢车间操作员、工艺工程师、生产调度员、质量检验员
- 产品价值：提升生产管理效率，保障工艺执行规范性，实现产品质量可追溯

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 操作员 | 数据录入、生产状态查看、工艺参数记录 |
| 工艺工程师 | 工艺制度配置、参数调整、报表查看 |
| 生产调度员 | 生产计划查看、进度监控、异常报警 |
| 质量检验员 | 性能检验记录、质量数据查看、不合格品处理 |

### 2.2 功能模块

1. **看板总览**：生产实时概览、关键指标展示、设备状态监控
2. **板坯入炉模块**：板坯入炉排序管理、板坯信息录入
3. **加热炉模块**：加热炉温制度、出炉温度记录、加热状态监控
4. **粗轧除鳞模块**：高压水除鳞记录、粗轧道次压下参数
5. **精轧机组模块**：精轧终轧温度、带钢厚度检测、板形凸度记录
6. **层流冷却模块**：层流冷却制度、冷却参数监控
7. **卷取打捆模块**：卷取温度控制、成品打捆称重
8. **性能检验模块**：力学性能取样、检验结果记录、质量报告

## 3. 核心流程

板坯经入炉排序后进入加热炉，按炉温制度加热至目标温度出炉；出炉后经高压水除鳞进入粗轧机组，按道次压下规程进行多道次轧制；粗轧后中间坯进入精轧机组，控制终轧温度与带钢厚度、板形；精轧后带钢经层流冷却至目标卷取温度；最后由卷取机卷取成卷，打捆称重后送性能检验，完成全流程。

    G --> H["成品入库"]

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深钢蓝 `#1e3a5f`（工业稳重感），辅以暖橙 `#e86a2c`（高温警示）、安全绿 `#2ea043`（正常状态）
- **辅助色**：钢灰 `#6b7280`、警示红 `#dc2626`、数据蓝 `#3b82f6`
- **背景色**：深色主题，主背景 `#0f172a`，卡片背景 `#1e293b`，营造工业控制室氛围
- **按钮风格**：直角或小圆角（2-4px），硬朗工业风格，hover有轻微阴影
- **字体**：中文字体使用思源黑体，数字等宽使用 JetBrains Mono，体现精密感
- **布局风格**：左侧导航栏 + 顶部面包屑 + 主内容区的经典后台布局，内容采用卡片式分区
- **图标风格**：使用 lucide-react 线性图标，统一线宽 2px

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 看板总览 | 生产概览 | 大型KPI数字卡片（带趋势箭头）、设备状态指示灯、工序进度条 |
| 看板总览 | 趋势图表 | 多Y轴实时曲线图（温度/压力/厚度）、数据刷新动画 |
| 各业务模块 | 数据表格 | 斑马纹、行悬浮高亮、状态标签多色区分、固定表头 |
| 各业务模块 | 参数表单 | 左右两列布局、标签左对齐、数值输入框带单位标注 |
| 各业务模块 | 状态监控 | 仪表盘组件、进度条、实时数据闪烁效果（超限时） |

### 4.3 响应式

- 桌面端优先（1440px+），适配工业现场大屏显示器
- 侧边栏固定宽度240px，主内容区自适应
- 表格区域支持水平滚动，适配多列数据展示
- 触摸屏适配：按钮最小尺寸40x40px

### 4.4 数据可视化

- 温度曲线：实时折线图，多段温度区间分色显示
- 厚度检测：带状图展示头中尾厚度偏差
- 设备状态：环形进度图+状态色标
- KPI指标：数字翻牌动画效果
